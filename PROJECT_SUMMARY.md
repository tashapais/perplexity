## 🚀 Getting Started

### Quick Start
```bash
# Clone and setup
git clone <repository>
cd researchhub
chmod +x setup.sh
./setup.sh

# Start services
npm run dev          # Frontend (localhost:3000)
npm run backend      # Backend (localhost:8000)
```

### Docker Deployment
```bash
docker-compose up --build
```
